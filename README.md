## 📌 نظرة عامة

**Buy Projects Pro** هو بوت Discord احترافي مصمم لـ **بيع البروجكتات بشكل تلقائي 100%** دون الحاجة لأي تدخل يدوي. يستخدم البوت نظام تذاكر متكامل، ويعتمد على **ProBot** لإتمام عمليات الدفع بالكريديت بأمان وسرعة.

> **⚠️ ملاحظة:** هذا البوت **خاص ومرخص** — يعمل فقط في السيرفرات المصرح لها من المطور. للحصول على نسخة خاصة بك، تواصل عبر [سيرفر الدعم](https://discord.gg/x2QqPMP9WB).

## ⚙️ كيف يعمل البوت

### مخطط العمل

```
المستخدم يضغط "🎫 فتح تذكرة شراء"
             ↓
 يُنشئ البوت قناة خاصة تلقائياً
             ↓
 المستخدم يختار المشروع من القائمة
             ↓
 يعرض البوت أمر التحويل مع المبلغ الكامل
      (السعر + 5% رسوم ProBot)
             ↓
 المستخدم يحول الكريديت عبر ProBot
             ↓
 يتحقق البوت من التحويل تلقائياً ✅
             ↓
 يُرسل رابط المشروع للمشتري فوراً 🎉
             ↓
 تُغلق التذكرة تلقائياً بعد 5 دقائق
```

### آلية التحقق من الدفع

- يراقب البوت رسائل **ProBot** في قناة التذكرة
- يتحقق من أن المحوَّل إليه هو البنك الصحيح
- يحسب تلقائياً رسوم 5% ويقبل المبلغ الصحيح
- مدة الانتظار: **5 دقائق** قبل انتهاء التذكرة

### آليات الأمان

- **كشف الحسابات المتعددة:** منع فتح أكثر من تذكرة واحدة بنفس الوقت
- **الكولدوان:** 15 دقيقة انتظار عند الإلغاء أو الفشل
- **إغلاق النشاط:** تُغلق التذاكر بعد 15 دقيقة من عدم الرد
- **حماية الترخيص:** البوت يغادر أي سيرفر غير مرخص فوراً

## 🎮 الأوامر

> **البادئة:** `+` (قابلة للتغيير في `config.json`)

## 🚀 دليل الإعداد

### المتطلبات

- **Node.js** v18.0.0 أو أحدث
- **npm** (يأتي مع Node.js)
- **توكن بوت Discord** من [Discord Developer Portal](https://discord.com/developers/applications)
- **ProBot** مضاف في سيرفرك مع صلاحيات كافية
- حساب ProBot مُعد كـ **بنك** لاستقبال الكريديت

### التثبيت

```bash
# 1. فك ضغط الملفات
unzip BuyProjects_Updated.zip
cd bot_modified

# 2. تثبيت المكتبات
npm install

# 3. إعداد الملفات (انظر قسم الإعداد)
# عدّل config.json و projects.js

# 4. تشغيل البوت
node index.js

# للتشغيل المستمر (يُوصى به):
npm install -g pm2
pm2 start index.js --name "BuyProjects"
pm2 save
```

### الإعداد

#### `config.json` — الإعدادات الأساسية

```json
{
  "token": "توكن البوت من Discord Developer Portal",
  "prefix": "+",
  "serverid": "ID سيرفرك",
  "allowedGuilds": ["ID سيرفرك"],
  "bankid": "ID حساب ProBot البنك لاستقبال الكريديت",
  "probotid": "282859044593598464",
  "clientrole": "ID رتبة العميل (تُضاف بعد الشراء)",
  "parent": "ID الكاتيغوري الذي ستُنشأ فيه التذاكر",
  "channellog": "ID قناة تسجيل العمليات",
  "reviewChannel": "ID قناة التقييمات",
  "VoiceID": "ID قناة صوتية للأبتايم",
  "SupportID": "ID رتبة الدعم الفني",
  "supportServer": "https://discord.gg/x2QqPMP9WB",
  "dev": ["ID خاصتك"]
}
```

#### `projects.js` — قائمة المشاريع وأسعارها

```js
module.exports = [
  {
    name: "اسم البروجكت الأول",
    url: "https://رابط-البروجكت.com",
    price: 1000   // السعر بالكريديت (البوت يحسب رسوم 5% تلقائياً)
  },
  {
    name: "اسم البروجكت الثاني",
    url: "https://رابط-البروجكت-2.com",
    price: 2000
  },
  // أضف المزيد هنا...
];
```

> **💡 ملاحظة الأسعار:** `price` هو المبلغ الذي **سيصل** إليك. البوت يحسب تلقائياً المبلغ الذي يجب على المستخدم تحويله بعد إضافة رسوم ProBot (5%).  
> **مثال:** سعر `1000` → المستخدم يحوّل `1053` كريديت → يصل إليك `1000`

### إعداد ProBot

1. أضف **ProBot** لسيرفرك من [probot.io](https://probot.io)
2. اذهب إلى إعدادات ProBot → Economy → فعّل الاقتصاد
3. احصل على **ID حساب البنك** (المستخدم الذي سيستقبل الكريديت)
4. ضع هذا الـ ID في `bankid` بملف `config.json`

## ❓ الأسئلة الشائعة

**س: لماذا يجب تحويل مبلغ أكبر من السعر المعروض؟**  
ج: ProBot يأخذ **5% رسوم** من كل تحويل. البوت يحسب المبلغ الصحيح تلقائياً ليصلك المبلغ الكامل.

**س: ماذا يحدث إذا حوّل المستخدم مبلغاً خاطئاً؟**  
ج: يرفض البوت التحويل ويعطي المستخدم تعليمات واضحة لتصحيح المبلغ، ثم تُغلق التذكرة بعد 5 دقائق.

**س: ماذا يحدث إذا كانت رسائل المستخدم الخاصة مغلقة؟**  
ج: يعرض البوت رابط المشروع مباشرة في قناة التذكرة بدلاً من الرسائل الخاصة.

**س: كيف أضيف سيرفراً جديداً مرخصاً؟**  
ج: أضف ID السيرفر لمصفوفة `allowedGuilds` في `config.json` ثم أعد تشغيل البوت.

**س: كيف أوقف تذكرة نشطة؟**  
ج: يمكن إغلاقها بالضغط على زر **🗑️ إغلاق التذكرة** داخل قناة التذكرة.

**س: هل يمكنني تغيير البادئة؟**  
ج: نعم، غيّر قيمة `prefix` في `config.json`.
